# 모터 전류 데이터 기반 이상 탐지 (STFT + CNN Autoencoder)

이 프로젝트는 모터에서 측정된 3상 전류 데이터(.mat)를 STFT(Short-Time Fourier Transform)를 통해 주파수 스펙트로그램 이미지 텐서로 변환하고, CNN 오토인코더(Autoencoder)를 학습시켜 비정상(고장) 상태를 탐지하는 딥러닝 파이프라인입니다.

## 📂 주요 폴더 구조 및 역할

*   **`normal_data/`** : 정상 상태에서 측정된 원본 모터 3상 전류 데이터 (`.mat` 파일)가 보관되는 폴더입니다.
*   **`abnormal_data/`** : 고장(비정상) 상태에서 측정된 원본 모터 3상 전류 데이터 (`.mat` 파일)가 보관되는 폴더입니다.
*   **`normal_image/`** : `STFT.ipynb` 코드를 거쳐 정상 데이터가 딥러닝용 텐서(`.npy`)로 변환되어 저장되는 폴더입니다. (오토인코더 **학습**용)
*   **`abnormal_image/`** : `STFT.ipynb` 코드를 거쳐 비정상 데이터가 딥러닝용 텐서(`.npy`)로 변환되어 저장되는 폴더입니다. (오토인코더 **테스트/검증**용)

---

## 🚀 실행 흐름 (Workflow)

프로젝트는 크게 **[데이터 전처리]** 단계와 **[모델 학습 및 이상 탐지]** 단계로 나뉘어 실행됩니다. 반드시 1단계 완료 후 2단계를 진행해야 합니다.

### 단계 1: 데이터 전처리 (`STFT.ipynb`)
원시 전류 데이터를 모델이 이해할 수 있는 형태(Numpy 배열)로 깎아내는 과정입니다.

1. Jupyter 환경에서 **`STFT.ipynb`** 파일을 엽니다.
2. 위에서부터 순서대로 셀을 실행(Run)합니다.
3. 스크립트가 `normal_data/`와 `abnormal_data/` 내부의 `.mat` 파일들을 읽어들입니다.
4. 신호를 1초 단위 조각(Chunk)으로 자르고, STFT 변환을 거쳐 주파수-시간 영역의 데이터로 만듭니다. (진폭의 왜곡을 막기 위해 모든 데이터는 **글로벌 최댓값/최솟값 기준**으로 절대 정규화됩니다.)
5. 처리가 완료된 텐서 파일들은 각각 `normal_image/`와 `abnormal_image/` 폴더에 `.npy` 확장자로 저장됩니다.

### 단계 2: 모델 학습 및 이상 탐지 (`Auto_encoder.ipynb`)
가공된 데이터를 바탕으로 정상 패턴만을 학습한 뒤, 비정상 데이터를 솎아내는 과정입니다.

1. 데이터 추출이 완벽히 끝난 후, **`Auto_encoder.ipynb`** 파일을 엽니다.
2. 위에서부터 순서대로 셀을 실행(Run)합니다.
3. **모델 학습:** `normal_image/`의 정상 데이터만 모델에 입력합니다. CNN 오토인코더는 병목(Bottleneck) 구조를 통해 공간 정보를 파괴한 뒤 필수 특징만으로 뼈대를 다시 복원하는 훈련을 합니다.
4. **임계값 설정:** 정상 데이터를 복원할 때 발생하는 픽셀의 최대 오차(Max MSE)를 계산하고, 여기에 약간의 여유를 두어 고장 판별 기준선(Threshold)을 확립합니다.
5. **이상 탐지:** `abnormal_image/`의 고장 데이터를 모델에 넣습니다. 모델은 본 적 없는 고장 주파수를 복원할 능력이 없기 때문에 픽셀 복원 오차가 임계값을 뚫고 치솟으며, 이를 통해 이상(Anomaly)으로 최종 판별합니다.
