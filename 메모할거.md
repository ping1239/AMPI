1. 변수 목록 (Variable Name)
이 파일을 MATLAB이나 Python(예: scipy.io.loadmat)으로 불러올 경우, 파일명과 동일한 최상위 구조체(Struct) 변수 하나가 반환되며, 그 안에 측정 정보와 센서 데이터가 계층적으로 포함되어 있습니다.
    N09_M07_F10_K001_1 (최상위 변수)
        Info: 소프트웨어 리비전(RevisionMajor, RevisionMinor), 측정 ID, 측정 일시(17.09.2015 19:51:22) 등의 메타데이터
        X: 시간 축(Time-axis)에 대한 파라미터 정보 (시작 시간, Time limit 등)
        Y: 센서별 실제 시계열 측정 데이터들이 담긴 하위 배열
            Name: 측정된 센서 및 신호의 이름 (모터 상전류, 진동, 토크, 속도 등)
            Type: 신호의 데이터 타입
            Data: 센서에서 수집된 실제 시계열 측정값
            Unit: 측정 단위 (A, m/s^2, rpm 등)
            Raster: 샘플링 속도 및 해상도 정보


2. 데이터 타입 및 크기 (Data Types and Dimensions)
바이너리 구조상 각 변수는 다음과 같은 형태와 크기를 띱니다.
    N09_M07_F10_K001_1: struct (1x1 구조체)
        Info, X: struct (1x1 구조체)
        Y: struct array (보통 1x7 내외의 크기. 여러 센서 채널 수에 해당함)
            Y(i).Name, Y(i).Unit: char (문자열 배열)
            Y(i).Data: double (부동소수점 실수 1차원 배열)
배열 크기 추정: 파일 내부 메타데이터에 측정 시간(Time limit)이 4초(4.000000)로 기록되어 있습니다. 첨부된 측정 로그에 따르면 전류(Current)와 진동(Vibration)의 샘플링 주파수는 64 kHz이므로, 이 데이터들의 크기는 256,000 x 1 (64,000 * 4) 입니다. 속도(Speed)나 토크(Torque) 같은 기계적 파라미터는 4 kHz이므로 16,000 x 1의 크기를 갖습니다.

3. 데이터 요약 (Data Summary)
    파일 이름(N09_M07_F10_K001_1)과 헤더에 기록된 정보를 바탕으로 데이터의 특성을 요약할 수 있습니다.
    측정 환경 요약:
        회전 속도: 900 rpm (N09)
        부하 토크: 0.7 Nm (M07)
        반경 방향 힘: 1000 N (F10)
        베어링 상태: K001 베어링에 대한 1번 테스트 런
    Y.Data 내부 값의 특성 (예상 값):
        진동 신호 (Vibration): 평균이 0에 가까운 교류(AC) 파형입니다. 베어링의 기계적 구동으로 인해 발생하는 고주파 충격파와 노이즈가 섞인 형태를 띠며, 최솟값과 최댓값은 수~수십 단위 범위 내에서 빠르게 진동합니다.
        3상 모터 전류 (Motor Current): 평균이 0 A에 가까운 안정적인 사인파(Sine wave) 형태입니다. 0.7 Nm의 부하를 유지하기 위한 특정 진폭 안에서 연속적인 double 형 실수 값(예: 0.0542, -0.0124...)들이 25만 개 이상 빽빽하게 배열되어 전류의 주기성을 나타냅니다.


K (예: K001 ~ K006): 정상 상태(Healthy / Reference)의 베어링입니다.
KA (예: KA01, KA05 등): 인위적으로 흠집을 낸(Artificially damaged) 고장 베어링입니다.
KI 또는 KB 등: 실제 자연적으로 마모되거나 손상된(Real damaged) 고장 베어링입니다.
